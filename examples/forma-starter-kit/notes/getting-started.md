# Getting Started

This example workspace lives under `examples/forma-starter-kit`. It can be
served directly by the Forma CLI.

## Check The Workspace

```sh
cargo run -p forma-cli -- --workspace examples/forma-starter-kit check
```

The check command validates configuration, spaces, pages, references, and
the summary index.

## Refresh The Index

```sh
cargo run -p forma-cli -- --workspace examples/forma-starter-kit index rebuild
```

Run this after editing example content. The summary index is generated by Forma
and should not be hand-edited.

## Serve The WebApp

```sh
cargo run -p forma-cli -- --workspace examples/forma-starter-kit serve
```

Open the printed local URL in your browser. The WebApp uses the same workspace
files that you can inspect in your editor.

## What To Edit First

Try changing the summary of [[notes/welcome-to-choral-forma|Welcome to Choral Forma]],
then rebuild the index and refresh the WebApp.
