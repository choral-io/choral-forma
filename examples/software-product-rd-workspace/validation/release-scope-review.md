# Release Scope Review

## Scenario

A reviewer opens [[releases/planning-beta]] and checks whether the release has enough linked scope and validation evidence to proceed.

## Expected Result

The reviewer can identify included tasks, validation cases, readiness metric, current blockers, and known deferrals without relying on chat history.
