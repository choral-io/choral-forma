# Readiness Review Checklist

This experiment asks whether a small checklist is enough to make release reviews repeatable without creating a heavyweight process.

## Method

For the planning beta, the team checks whether the release record links to scope tasks, validation cases, and readiness metrics before approval.

## Success Signal

The release reviewer can decide whether the release is ready within one review pass.
