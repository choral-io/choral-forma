# Minimal Forma Workspace

This example is the smallest committed Forma workspace. It shows one configured space, one create template, one saved view, and two connected Markdown entries.

Use this when you want to understand the bootstrap shape before adding richer spaces, schemas, views, or guidelines.

Verify it with:

```sh
forma --workspace examples/minimal-workspace config inspect --json
forma --workspace examples/minimal-workspace check --json
forma --workspace examples/minimal-workspace workspace health --json
```
