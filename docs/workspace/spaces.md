# Spaces

## Overview

A space is a configured group of Markdown entries. Forma does not have built-in task, note, member, or guideline concepts; those are ordinary spaces defined by configuration.

## Agent Guidance

Create a space only after the human describes a durable content category. Define include patterns, schema, create behavior, and guidelines explicitly.
