# Default Team

## Purpose

This group represents the default responsibility group for project knowledge ownership, fallback review, and workflow governance when no more specific member or group owner is assigned.

Use `[[groups/default-team]]` for tool-written shared ownership that belongs to the default group. Manual `[[default-team]]` references are acceptable when they resolve unambiguously to this group.
