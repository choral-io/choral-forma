# Editor-Independent Knowledge

## Definition

Editor-independent knowledge is repository knowledge that remains readable and maintainable across normal repository tools, text editors, and knowledge editors.

## Context

Choral Forma may support Foam, Obsidian, or future application interfaces, but project facts should not depend on an editor-specific plugin, cache, database, or syntax extension.
